Follow these instructions to run the project

1) Put zoho.xls file in Users/Desktop/Readingdata folder.

2) To download the supporting jars.
   - Open Terminal prompt
   - /Users/maven/DataDriven_zoho_2016
   - run pom.xml by typing mvn compile
   - mvn eclipse:eclipse

3) Import poi jars files from lib folder.

MUST HAVE Mozilla firefox ver 30.0
